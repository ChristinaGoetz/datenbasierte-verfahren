1. Melden Sie sich mit einem Google Konto an (z.B. durch Klick auf den Anmelden-Button oben links).

2. Erstellen Sie eine Kopie eines Google Colab Notebooks "DV_XXX.ipynb".

3. Öffnen Sie Ihre lokale Kopie des Google Colab Notebooks. 

4. Führen Sie die einzelnen Zellen des Notebooks (grau hinterlegte Blöcke) aus (z.B. durch Klick auf den Play-Button oben links in jedem grau hinterlegtem Block). Passen Sie die beschriebenen Stellen an.
